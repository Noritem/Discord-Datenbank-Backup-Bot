# Discord Backup Bot JOINE MEINEM DISCORD GERNE https://discord.gg/Xv8UynvgZp https://discord.gg/Xv8UynvgZp

Dieser Discord-Bot erstellt regelmäßig Backups einer MySQL-Datenbank und lädt sie in einen Discord-Kanal hoch.

## Installation

1. Stelle sicher, dass Python und pip auf deinem System installiert sind.

2. Navigiere in das Verzeichnis des Projekts.

3. Installiere die Abhängigkeiten, indem du den Befehl ausführst: python installer.py

4. Konfiguriere den Bot, indem du die `config.ini`-Datei bearbeitest und deine Discord-Bot-Token, 
MySQL-Verbindungsinformationen und andere Einstellungen angibst.

5. Starte den Discord-Bot, indem du den Befehl ausführst: python SimpleBackup_Bot.py

6. Der Bot wird gestartet und ist nun bereit. Er erstellt automatisch Backups der MySQL-Datenbank entsprechend der konfigurierten Backup-Zeitangabe.

7. Der Bot lädt jedes Backup in den angegebenen Discord-Kanal hoch.

## Hinweise

- Stelle sicher, dass der Discord-Bot auf deinem Server vorhanden ist und die erforderlichen Berechtigungen hat, um Nachrichten in den angegebenen Kanal hochzuladen.
- Überprüfe die Berechtigungen der MySQL-Datenbankbenutzer, um sicherzustellen, dass sie die erforderlichen Rechte zum Erstellen von Backups hat.
